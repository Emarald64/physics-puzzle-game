# Physics Puzzle Game

You are given a framework of blocks that you need to made your own additions to to prevent them from falling down. 

Place blocks to prevent any of the blocks from falling below the red line!

Block sprites from: https://kenney.nl/assets/physics-assets

Level tiles from: https://kenney.nl/assets/platformer-art-extended-tileset

Made with the Godot Engine


## To play:

Run the web version or download the desktop builds from the game's page on [itch.io](https://xanderath.itch.io/physics-puzzle)

or

1. Download Godot from https://godotengine.org/

2. Download the repo using
``git clone https://github.com/Emarald64/physics-puzzle-game.git``

4. Import the folder you cloned the repo into Godot
